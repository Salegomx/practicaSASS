Aprendiendo SASS

Comandos para ejecutar

  • npm init // Iniciar un proyecto con un archivo package.json
  • "sass": "sass --watch --style=compressed ./scss/style.scss ./css/style.css" // Script para observar los cambios en el archivo .scss
  • npm run sass // Se ejecuta el script y la consola comienza a observar los cambios

Para que sirve el archivo .map?

El archivo .map sirve para "mapear" o "rastrear" los cambios que se van haciendo en scss y css. Está destinado a entornos de desarrollo y no se debe incluir en entornos de produccion para aspi reducir el tamaño de recursos enviados al navegador
